Hardwares e softwares em servidores

Toda a evolução, conceito de criação dos serves e segurança envolvida.
A criação do proxy para definir as regras de comunicação entre Clients e Servers, Tinha muita curiosidade de como funcionava um site, por exemplo, onde ele ficava hospedado e como conseguir colocar ele no "ar". Uma das partes também que me interessei bastante foi para a criação de banco de dados dentro dos servidores, temos que utilizar um sistema operacional para gerenciar toda essa comunicação com o hardware.

O que sao Clients

É basicamente oque o usuário solicita ao servidor, sendo uma troca de informações, o usuário solicita algo ao servidor e o servidor deve retornar como por exemplo uma imagem, aúdio, video ou qualquer outro tipo de arquivo.
Lembrando que deve ser solicitado por um tipo de browser ou app, sendo assim estabelecido uma conexão com o servidor.

Linguagem de programação

Vimos que para servidores temos um tipo de linguagem e para app's ou browser temos outros tipos de linguagens, que algumas somente, que se comunicam entre as 2 frentes.
Também entendi que HTML não é uma linguagem de programação e sim um "arquivo de marcação", ou seja, um tipo de texto que os navegadores conseguem interpretar....

Surgimento da internet

Da pra notar que a internet surgiu não para a finalidade que vemos hoje, é muito superficial pensar que todo meio de comunicação foi feito para a população em geral de modo acessível, isso foi desenvolvido para comunicação, porém, aprimorado somente após a primeira guerra.
Os primeiros computadores gigantescos que na época não faziam coisas que hoje um relógio de pulso faz sem se quer esforço e uma internet que "pingava" para fazer um download.
